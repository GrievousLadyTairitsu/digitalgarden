##### Naming Rule Orders
- 1. Parent Chain
	- Prioritize simpler branches over more complex branches
		- Ie. Double bonds > triple bond
- 2. Type of branch
	- Greek prefix (num of a type of bonds) $\ne$ count in alphabetical order 
- 3. Location of each branch must be as low as possible
	- Order of branches are based off alphabetical order
- Tiebreaker rule
	- For two branch types that share the same listing order, alphabetical order rules
	- ie. two diff types of branches both share a 4th spot in a 8 carbon chain
