# Proof of the Mean Value Theorem for integrals
Recall: **Mean Value Theorem**
If the function $\mathbb f$ is **continuous** on the **closed interval** $\mathbb [a,b]$ and **differentiable** on the **open interval** $\mathbb (a,b)$, there **exists** a value c on $\mathbb [a,b]$ such that:
$$ f'(c)=\frac{f(b)-f(a)}{b-a}

$$
Then:
$$f'(c)(b-a)=f(b)-f(a)$$
Recall: Definition of the **area under a curve** using Riemann Sums
If a function $\mathbb f$ is **continuous** on the **closed interval** $\mathbb [a,b]$, then the **area** under the curve wrt to the x-axis is 
$$\lim_{n \rightarrow \infty} \sum_{i=1}^{n}f(x_i)\Delta x$$
$$\Delta x = \frac{b-a}{n} \ , \ x_i \in [i, i-1]$$
$$\mathbb f(b)-f(a)= \lim_{n \rightarrow \infty} \sum_{i=1}^{n}[f(x_i)-f(x_{i-1})]\ , \ x_0=a$$
