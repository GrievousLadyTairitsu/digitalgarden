# Distance Position Displacement
Key Terms:
- Entity: Anything measurable scientifically
- Scalars: Magnitude only entity, eg. Mass, Temp(C)
- Vectors: Magnitude and Directional Entity
#### Distance (d):
- Scalar unit
- Path Length of function given t is defined and restricted as $t\in{[x,y]}$
#### Position ($\vec{r}$)
- Vector Unit
- Distance and direction from reference point ($\vec{r_0}$)
#### Displacement ($\Delta \vec{r}$)
- Vector unit
- Change in positions with respect to ($\vec{r_o}$)
	- $\Delta \vec{r} = \vec{r_a} - \vec{r_b}$
#### Total Displacement ($\Delta \vec{x_t}$)
- Total of all defined displacements (n-1) of all positions (n)
	$$\mathbb \Delta \vec{x_t} = \sum_{z=0}^{n-1} \Delta \vec{x_{n-z}}$$
#### Resultant Displacement ($\Delta \vec{x_r}$)
- Difference between final defined position and reference point:
	- $\Delta \vec{x_r} = \vec{x_n} - \vec{x_0}$
