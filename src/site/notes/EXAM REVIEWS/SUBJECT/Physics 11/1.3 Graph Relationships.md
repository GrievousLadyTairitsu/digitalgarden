#  Uniformity and Acceleration
#### Uniform Motion
- Motion at a constant velocity
- Covers a consistent displacement regardless of time period (give domain is restricted)
#### Non-Uniform Motion
- Velocity changes in the $\vec{v}(t)$ function given there is a restricted domain
- Changes speed of object in a direction
#### Acceleration ($\vec{a}$)
- Rate of change in velocity given there is a restricted domain
	- $\vec{a}=\frac{\Delta \vec{v}}{\Delta \vec{t}}$ 
- **Uniform Accelerated Motion** defines $\vec{a}$ as a constant
#### Graph Relationships
- Derivative of $\vec{r} \ vs \ t$ graph describes the $\vec{v} \ vs \ t$ graph
- Derivative of $\vec{v} \ vs \ t$ graph describes the $\vec{a} \ vs \ t$ graph
- Area of $\vec{v} \ vs \ t$ graph describes $\Delta \vec{r}$ given t is in a restricted domain
	- $$A(t)= \int_0^t \vec{r}(t)dt$$
- Area of $\vec{a} \ vs \ t$ graph describes $\Delta \vec{v}$ given t is in a restricted domain
	- $$A(t)= \int_0^t \vec{v}(t)dt$$
