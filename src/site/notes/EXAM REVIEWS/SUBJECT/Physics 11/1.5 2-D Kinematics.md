# 2-D Kinematics
#### Composition Method
- Composition Method outlines a method that takes calculations of x and y values separately, all connected through the concept of time :O. 
- Here's how it can be done:
	- The initial velocity ($\vec{v_i}$) of object x is A m/s at 30 degrees.
	- We can identity the x and y components through cosine and sine.
		- $\vec{v_{i_x}}$ = $\vec{v_i} \ cos(30^{\circ})$  
		- $\vec{v_{i_y}}$ = $\vec{v_i} \ sin(30^{\circ})$  
	 - In addition, this can also be applied to the concept of acceleration
		 - If we know that $\vec{v_{f_x}}$ = C m/s, then it can be inferred that a $\vec{v_{f_x}}$ = $\vec{v_{i_x}} + \vec{a_x} \Delta t$ 
		 - The same can concur for y: $\vec{v_{f_y}}$ = $\vec{v_{i_y}} + \vec{a_y} \Delta t$ 
	- Notice that theres no degrees in the x and y sections?
		- Thats the power of **composition**, which enables this 
- Calculating Acceleration and Pythagoras
	- If we can calculate only $\vec{a_x}$ or $\vec{a_y}$ , then we are severely limited. Theres a easy way to deal with this however... 
		- $\vec{a} = \sqrt{\vec{a_x}+\vec{a_y}} \ [\theta_a]$ given that $[\theta_a]=tan^{-1} (\frac{\vec{a_y}}{a_x})$ 
	- This concept can be applied to ALL of kinematics!
